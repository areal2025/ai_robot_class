📋 교과목 소개

본 교과목은人工智能과 로봇 기술을 결합한 실용적인 교육을 제공합니다. 학생들은 Python 프로그래밍 기초부터 시작하여 ROS2 로봇 운영 체제, 그리고 AI 어시스턴트와의 연동까지 단계별로 학습하게 됩니다. 특히 음성 명령이나 메시지로 로봇을 제어하는 시스템을 직접 구현해 보며, 실제 산업 현장에서 활용되는 기술들을 경험할 수 있습니다. PyBullet 기반의 웹 시뮬레이션을 활용하여 하드웨어 없이도 실습이 가능하고, 팀 프로젝트를 통해 종합적인 시스템 설계 역량을培养합니다. 본 과정을 통해 학생들은 로봇 공학의 기초 개념을 이해하고, AI 기술을 활용한创新적인解决方案을 제시할 수 있는 역량을 갖출 수 있습니다.

2. 교육목표 (Course Objectives)

(1) 기초 이해

로봇 공학, AI, ROS2의 기본 개념과 원리를 이해한다.

(2) 프로그래밍 역량

Python 프로그래밍 능력을 갖추고 로봇 제어 코드를 작성할 수 있다.

(3) 시스템 구현

AI 어시스턴트와 ROS2 로봇을 연동하여 음성/메시지로 로봇을 제어하는 시스템을 구현한다.

(4) 문제 해결

실제 로봇 문제에 대해 분석하고 해결책을 설계 및 구현할 수 있다.

(5) 프로젝트 수행

팀 프로젝트를 통해 종합적인 로봇 시스템을 설계하고 구현할 수 있다.

3주차 Python 프로그래밍 Python 기초 문법, ROS2 Python 라이브러리 활용 Python으로 간단한 로봇 제어 코드 작성

6주차 폐루프 제어 PID 제어, 속도 제어, 모터 제어 기초 로봇의闭环 제어 시스템 구현
7주차 중간 복습 1~6주차 전체 내용 복습 및 실습 전반적인 지식 정리 및 취약부분 보완
8주차 중간고사 필기 시험 (공식, 개념, 계산) 이론적 지식 평가 양면 A4 2장 지참 가능
9주차 음성 상호작용 STT/TTS, ROS2 음성 제어 노드 구현 음성 명령으로 로봇 제어
10주차 제스처 및体感 제어 손동작 인식, 제스처 인식 기초 제스처로 로봇 명령
11주차 운동 경로 계획 A* 알고리즘, 경로 계획, ROS2 내비게이션 자율주행 로트의 경로 계획 이해
12주차 시각 인식 이미지 처리, 물체 검출, OpenCV 기초 카메라 기반 시각 인식 시스템 구현
13주차 팀 프로젝트 I 프로젝트 구현 및 디버깅 팀 프로젝트 진행 프로젝트 디버깅 지도
14주차 팀 프로젝트 II 프로젝트 완성 및 데모 최종 프로젝트 완성 데모 Day
15주차 기말고사 필기 시험 (공식, 개념, 계산) 학기 전체 지식 평가 양면 A4 2장 지참 가능
