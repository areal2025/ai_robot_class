## 1.3 机器人与AI的区别与联系（10分钟）

### 🤖 什么是机器人？

> **机器人**（Robot）是一种能够自动执行任务的**物理机器**，它通常具有：
> - 传感器（感知环境）
> - 执行器（执行动作）
> - 控制器（处理信息）

### 🧠 什么是人工智能？

> **人工智能**（Artificial Intelligence, AI）是计算机科学的一个分支，致力于开发能够**模拟、延伸和扩展人类智能**的理论、方法、技术和应用系统。

### 2.2.1 为什么选择WSL2？

```
┌─────────────────────────────────────────────────────────────────┐
│                    WSL2 优势                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2.2 WSL2 + Ubuntu + ROS2 一键安装

> 最简单的安装方式，使用一键脚本：

```bash
# 1. 以管理员身份打开PowerShell

# 2. 安装WSL2（如果没有）
wsl --install

# 3. 打开Ubuntu终端，运行一键安装脚本
wget http://fishros.com/install -O fishros
bash fishros
```

> 📌 选择 **1. 一键安装** 即可自动完成ROS2安装！

### 2.2.3 手动安装（备选）

如果一键安装失败，可以手动安装：

```bash
# Ubuntu 22.04
sudo apt update
sudo apt install ros-humble-desktop

# 环境配置
source /opt/ros/humble/setup.bash
```

### 2.2.4 验证ROS2安装

```bash
# 启动小乌龟
ros2 run turtlesim turtlesim_node

# 另一个终端启动控制
ros2 run turtlesim turtle_teleop_key
```

## 2.3 OpenCLAW 环境搭建（15分钟）

### 2.3.1 什么是OpenCLAW？

> OpenCLAW是一个开源的AI助手框架，可以将AI大模型与机器人系统连接，实现自然语言控制。

**功能特点：**
- 🎙️ 支持语音控制
- 💬 支持消息控制（飞书/微信/Telegram）
- 🤖 支持ROS2机器人
- 🔌 支持多种AI模型

### 2.3.2 安装OpenCLAW

**方式一：Docker安装（推荐）**

```bash
# 拉取镜像
docker pull openclaw/openclaw:latest

# 运行
docker run -d -p 8080:8080 openclaw/openclaw
```

**方式二：pip安装**

```bash
pip install openclaw
openclaw server
```

### 2.3.3 连接飞书机器人

> 免费Token连接，无需付费！

1. **创建飞书应用**
   - 打开 https://open.feishu.cn/
   - 创建企业自建应用
   - 添加机器人能力

2. **获取凭证**
   - App ID
   - App Secret
   - Encrypt Key（用于验证）

3. **配置OpenCLAW**
   ```bash
   # 编辑配置文件
   nano ~/.openclaw/config.yaml
   ```
   
   ```yaml
   feishu:
     app_id: "your_app_id"
     app_secret: "your_app_secret"
     encrypt_key: "your_encrypt_key"
   
   ros2:
     enabled: true
     domain_id: 0
   ```

### 2.3.4 测试OpenCLAW

```bash
# 启动服务
openclaw server

# 在飞书发送消息测试
# "你好" → 应该收到自动回复
```
